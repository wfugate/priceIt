## How to run
### When in the backend directory
- run `ngroc http 5238` --> ngroc will give you a link that looks similar to this: `...`
- Change all links in `index.tsx` and scan `service.ts`

### When in the frontend directory
- run: `npm install`
- cd into `cs392app`
- run: `npm expo start`
